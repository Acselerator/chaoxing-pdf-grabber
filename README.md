# 学习通 PDF 抓取助手

一个用于学习通/超星课程章节页的 Tampermonkey/Violentmonkey 用户脚本。脚本会在课程章节树中添加下载按钮，帮助提取课程中可访问的 PDF 课件直链，并支持按小节直接下载、按章节生成 ZIP、按全课生成 ZIP。

## 功能

- 在课程章节页顶部添加“全课 PDF ZIP”按钮。
- 在每个大章节行添加“本章 ZIP”按钮。
- 在每个小节或中间章节行添加“PDF”按钮。
- ZIP 使用 STORE 方式写入，避免浏览器端进行耗时的 PDF 压缩计算。
- 大体积 ZIP 会先弹出风险提示，可选择导出直链清单或继续生成。
- ZIP 内保留课程/章节目录结构，解压到当前文件夹时不容易散落文件。

## 安装

推荐从 Greasy Fork 安装。发布后可将这里替换为 Greasy Fork 脚本地址：

```text
https://greasyfork.org/scripts/你的脚本编号
```

也可以在 Tampermonkey/Violentmonkey 中新建脚本，将 `chaoxing-pdf-grabber.user.js` 的内容粘贴进去。

## 使用

1. 打开学习通课程章节页。
2. 等待章节树加载完成。
3. 根据需要点击：
   - `全课 PDF ZIP`
   - `本章 ZIP`
   - `PDF`
4. 如果 ZIP 体积过大，脚本会提示浏览器卡顿或崩溃风险；可以取消、导出直链清单，或继续生成。

## 适用范围

脚本不绑定特定课程、班级或学校。它会从当前页面动态读取课程 ID、班级 ID、章节 ID 等信息。

理论上，只要课程使用学习通/超星相同或相近的章节页结构，并且当前账号有权限访问对应 PDF，脚本就有机会工作。

已知限制：

- 只处理可解析到的 PDF 课件，不保证支持视频、Word、PPT、作业附件等其他资源。
- 如果课程页面结构、接口参数或资源权限策略不同，部分课程可能无法解析。
- 跨校、跨课程是否可用取决于对方课程是否仍使用相同的学习通页面和资源接口。
- 生成大体积 ZIP 会占用浏览器内存，极端情况下可能导致标签页卡顿或浏览器崩溃。
