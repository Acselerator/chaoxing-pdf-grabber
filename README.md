# 学习通 PDF 抓取助手 (Chaoxing PDF Grabber)

[![Greasy Fork](https://img.shields.io/badge/Greasy%20Fork-v1.0.0-blue.svg)](https://greasyfork.org/zh-CN/scripts/577167)
[![JavaScript](https://img.shields.io/badge/Language-JavaScript-yellow.svg)]()
[![License](https://img.shields.io/badge/license-MIT-green)](./LICENSE)

一个运行在浏览器端的 **Tampermonkey / Violentmonkey** 用户脚本。采用纯前端架构，无需配置环境，无服务器参与。一键提取学习通/超星课程中可访问的高清 PDF 课件直链，并支持将全课或单独章节课件无损打包为 ZIP 下载，为您带来绝对安全、极简的离线学习体验。

---

## 📖 项目背景

在许多学习通课程中，教学课件（如 PPT/PDF）往往被直接嵌入在课程章节页面里，官方并未提供便捷的批量下载按钮。这给离线阅读、资料整理以及后续的知识管理带来了极大的不便。

<img width="1920" height="1032" alt="痛点截图" src="https://github.com/user-attachments/assets/5ddb03f3-fb01-4b1e-adec-e72f70059f62" />

传统的解决方案多基于后端爬虫（如 Python），不仅需要用户配置复杂的运行环境，还需要提取敏感的 Cookie，存在隐私泄露风险。同时，部分传统方案采用“截图拼接”法，导致保存的 PDF 画质受损且文字无法复制。

**本项目** 旨在完美解决这一痛点：通过在页面注入脚本，利用用户的真实登录态，拦截后端 API 直接提取原始 CDN 高清物理直链。整个过程在您的浏览器内存中全自动化完成，**绝不触碰或收集用户的 Cookie 等隐私数据**，让高清无损的知识触手可及。

## ✨ 核心特性

- **🚀 纯前端零配置**：开箱即用，安装油猴插件即可，打通学习通的“任督二脉”，无需安装 Node.js 或 Python 等后端环境。
- **🔒 绝对隐私安全**：所有请求均在本地浏览器发起，利用您当前的合法登录态获取数据，不涉及任何第三方中转服务器及 Cookie 转移。
- **💎 高清无损提取**：直接获取官方 CDN 原始 PDF 直链，拒绝画质受损的截图拼接，100% 保留原本质量与文本可复制性。
- **🗂️ 一键智能打包**：支持按小节下载、按大章生成 ZIP、按全课生成 ZIP。ZIP 内自动生成嵌套的课程/章节目录结构，解压不凌乱。
- **🛡️ OOM 内存防崩机制**：内置超大体积打包防崩溃机制。对于体积超大（例如 >200MB）的课件合集，脚本会智能预判并拦截，提供“**导出直链清单 (.txt)**”的安全降级方案，避免浏览器卡顿或崩溃。

## 📦 安装说明

### 1. 安装脚本管理器
推荐使用以下任意一款浏览器扩展：
- [Tampermonkey (油猴)](https://www.tampermonkey.net/) (支持 Chrome, Edge, Safari, Firefox 等)
- [Violentmonkey (暴力猴)](https://violentmonkey.github.io/)

### 2. 安装本脚本
推荐直接从 Greasy Fork 安装，以获取自动更新及最佳体验：
👉 **[点击此处前往 Greasy Fork 安装](https://greasyfork.org/zh-CN/scripts/577167)**

> 也可以在脚本管理器中新建脚本，将本项目内 `chaoxing-pdf-grabber.user.js` 的源码复制粘贴进去并保存。

## 🚀 使用指南

1. **登录平台**：在浏览器中打开并登录您的学习通/超星账号。
2. **进入章节页**：进入任意需要提取课件的课程，切换到“文档”或“章节”目录页面。
3. **等待解析**：稍微等待章节树加载完成，脚本会自动分析 DOM 并在对应的章节行注入下载按钮。
4. **按需下载**：
   - 提取整门课：点击页面顶部 **`[全课 PDF ZIP]`**
   - 提取单章：点击大章标题右侧的 **`[本章 ZIP]`**
   - 提取具体小节：点击具体课件节点右侧的 **`[PDF]`**
5. **应对超大课件**：如果选择下载整课且总体积过大，脚本会弹出风险警告。您可以选择**继续强行打包生成 ZIP**，或者更安全地**导出直链下载清单 (.txt)**，再配合迅雷、IDM、Motrix 等第三方多线程下载工具进行批量下载。

## 💡 适用范围与边界限制

- **动态适配**：脚本不绑定特定课程或学校，通过页面特征动态读取 ID。只要对应课程使用标准版学习通页面结构，且**您当前账号具有在线浏览该 PDF 的权限**，脚本即可工作。
- **资源类型兼容限制**：脚本仅专注提取可被解析的 **PDF / 文档类** 资源。不保证全面支持或能够抓取视频、纯文本题库、作业附件、独立 PPT 文件等其他变体。部分特殊排版或被教师特别锁定权限的课程可能无法解析。
- **跨域与环境限制**：生成大体积 ZIP 使用 HTML5 `Blob` 与 `JSZip` 库，极大地吃内存。极端情况下会导致当前标签页卡顿甚至崩溃，此为浏览器本身的架构瓶颈。
